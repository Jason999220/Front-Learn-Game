首先跟各位解說一下CSS語法的撰寫方式
稍後會再詳細說明那些color、font-size等等到底是甚麼

介紹 CSS選擇器 文字 位置 特效 排版 小試身手 
            div { background-color : peru ;}

            div => 選擇器
            background-color => 屬性
            peru => 值

            css語法是由 => 選擇器{ 屬性 : 值 ; } 所構成的，屬性和值是以:來連接
          
            div { font-size : 16px ;}
            div { color : red ;}
            div { background-color : peru ;}

            如果有是針對相同的選擇器則可合併成一個
            但記得要在每個值的結束加上分號【;】

            div {
              font-size : 16px ;
              color : red ;
              background-color : peru ;
            }
          
            h2 {
              color : peru ;
            }
            h2 {
              color : red ;
            }

            假如選擇器有重複的屬性則會以權重來判定
            假如權重相同則會以程式碼的先後順序
            權重是什麼咚咚後續會討論到 ~~~
          
            h1 {color : peru ; }
            h2 {color : peru ; }
            p {color : peru ; }

            若有相同的屬性與值可以將其合併成同一條，切記選擇器要以逗號【,】後分開

            h1, h2, p {color : peru ; }
