打印出window，那window是甚麼呢？我又沒宣告怎麼能使用呢？對這邊就要提到另一個概念【全域變數】
簡單來說就是無論在哪個地方都可以呼叫這個變數，用var宣告的變數也會變成全域變數，展開window物件就可以看到剛剛用var定義的myName變數
這裡有兩個問題點
1. 假如我們不將變數宣告會怎樣呢？
2. 宣告後的變數與全域的變數有何不同

第一個問題點就像你操作的一樣，就算不宣告還是可以賦於值給變數，但此時的myName變數並不能說是全域變數，嚴格來說稱為window物件下的一個屬性。
                function callName(){
                  var myName = "Jason" ;
                }
                callName()
                console.log(myName)
