Boolean 是一個布林值，僅有True、False兩種值，之後很常用來判斷資料

JS

                var x = Symbol() ;
                var y = Symbol('Jason') ;

                console.log("x = ", x);
                console.log("y = ", y);
                console.log(y.description);

Console

              x = Symbol()
              y = Symbol(Jason)
              Jason
Symbol()的值是獨一無二的，可以用【description】屬性取得()內的資料
由於物件的key有可能會發生重複的情況導致資料被覆蓋，此時就可以使用Symbol()解決此問題看以下範例
var obj = {
[Symbol('Jason')] : {... },[Symbol('Jason')] : {...}
}

undefined 只有 undefined 這個值
當變數有宣告但沒有被賦予值的時候，它的值就會是undefined
要注意它與【not defined】不一樣，後者是變數未被宣告，不屬於原始型別，是一種程式執行時會出現的錯誤訊息

JS

                var a = [1,2] ;
                var b = {x:1,y:2} ;
                var c = function(){} ;

                console.log("a = ", a);
                console.log("b = ", b);
                console.log("c = ", c);

Console

              a = [1,2]
              b = {x:1,y:2}
              c = f(){}
接下來就換物件型別啦~~基本上除了以上的原始型別都可以稱作為物件型別。
此時的a變數稱為陣列是以方括號包起來，裡面可以放任何資料，可以有字串、數字、boolean等。
此時的b變數稱為物件是以花括號包起來，裡面是以【key:value】的形式儲存資料
此時的c變數稱為函式
以上這些會在稍後詳細說明，現在只需要知道它們為物件型別即可

JS

                var a = "Jason" ;
                var b = 5 ;
                var c = true ;
                var d = Symbol();
                var e = null ;
                var f  ;
                var g = [];
                var h = {};
                var i = function () {};

                console.log('typeof a = ', typeof a);
                console.log('typeof b = ', typeof b);
                console.log('typeof c = ', typeof c);
                console.log('typeof d = ', typeof d);
                console.log('typeof e = ', typeof e);
                console.log('typeof f = ', typeof f);
                console.log("typeof g = ", typeof g);
                console.log("typeof h = ", typeof h);
                console.log("typeof i = ", typeof i);

Console

              typeof a = string
              typeof b = number
              typeof c =boolean
              typeof d = symbol
              typeof e = object
              typeof f = undefined
              typeof g = object
              typeof h = object
              typeof i = function
可以使用【typeof 變數】檢查該變數的型別

各位可以想想看此時的變數均為什麼型別？
一定很好奇為甚麼null、function和自己不太一樣吧
null 是得到 object 而非 null，這是一個大bug，但若修正這個bug可能會導致許多網站壞掉，因此就不修了。
function是得到 function 而非 object，函式是一種「可呼叫的物件」讓它成為能夠被調用的物件。
